# Creando el proyecto Vite para el integrador


```sh
npm create vite@latest ./ -- --template vanilla
```

## Instalamos dependencias

```sh
npm i
```

## Arrancamos el servidor de desarrollo

```sh
npm run dev
```

## Instalando el procesador de SASS

```sh
npm i sass-embedded -D
```
